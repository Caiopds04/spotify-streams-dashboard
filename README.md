# spotify-streams-dashboard

Dashboard desenvolvido em Power BI para análise de streams do Spotify utilizando um dataset do Kaggle.

## Objetivo

Explorar dados de artistas e músicas populares através de visualizações interativas e indicadores de desempenho.

## KPIs

- Total Streams
- Melhor Posição Alcançada
- Média de Streams
- Maior Pico de Streams

## Visualizações

- Top Artistas por Streams
- Top Músicas por Streams
- Top 5 Artistas por Pico de Streams
- Filtro interativo por artista

## Ferramentas Utilizadas

- Power BI
- Kaggle
- Power Query
